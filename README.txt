CoinMarketCap Scraper

Description:
CoinMarketCap Scraper is a simple web scraping application that extracts information about the top cryptocurrencies and their current prices from the CoinMarketCap website.

How to Run:
To run the application, follow these steps:

1. Install the required libraries: pip install requests beautifulsoup4
2. Open the `main.py` file in a Python environment.
3. Run the script:
    python ./main.py
4. The application will connect to the CoinMarketCap website, scrape the data, and display the names and prices of the top 10 cryptocurrencies.
